# AngularMaterialTree

Sample project to use angular material tree component `v-6.3.0`

## Run the project

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`.

## Plug and Play

Copy paste the `tree` folder and import the `TreeModule` in your code
